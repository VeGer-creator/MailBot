# 📧 MailBot — Telegram бот для отправки файлов на email

Бот позволяет отправлять фото и документы из Telegram на указанный email адрес.  
Поддерживает групповую отправку, сохранение до 3 email адресов и автоматический перезапуск при зависании.

---

## 🚀 Быстрый старт

### 1. Установка
- Скачайте папку `MailBot_Desktop`
- Отредактируйте файл `.env` (вставьте свой токен и пароль)

### 2. Запуск
- **Простой запуск:** дважды кликните `start_bot.bat`
- **С авто-перезапуском:** дважды кликните `start_watchdog.bat`

---

## 📋 Команды бота

| Команда | Описание |
|---------|----------|
| `/start` | Начать работу, очистить все данные |
| `/emails` | Показать сохранённые email адреса (до 3) |
| `/clearemails` | Очистить список сохранённых email |
| `/stats` | Показать статистику (только для админа) |

---

## 📤 Как отправить файлы

1. Отправьте боту **фото** или **документы** (можно несколько сразу)
2. (Опционально) Напишите **тему письма** — отдельно или в подписи к файлу
3. Нажмите кнопку **"Готово"**
4. Выберите email из сохранённых или введите новый
5. Бот отправит письмо с файлами
